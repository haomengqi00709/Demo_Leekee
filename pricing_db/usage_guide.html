报价系统 · 怎么用

把原来销售翻表 约一小时、还容易出错的报价，变成 几秒钟。打开网址就能用，不用登录、不用装软件，电脑手机都行。

🔗
在浏览器打开我们发给你的网址，会看到顶部有三个标签：报价员 规则 数据。点标签切换。
报价员给客户出报价单 —— 平时用得最多
规则调加成 / 折扣 / 工艺费，说一句话就改
数据工厂发来新报价单，上传就进系统

1 报价员 · 出一张报价单

选套系 → 选产品 → 配工艺 → 加入报价单，几秒出价。
  1. 搜/选套系：最上面的搜索框里输套系号、产品编号或供应商（如 H563、CC20918、伟诚），或直接下拉选。
  2. 选产品、配工艺：选定具体产品后，下面每个部件（外罩 / 泵头 / 盖…）挑一档工艺（实色 / 电镀…），右侧价格实时算出来。
  3. 设数量、客户档、印刷：填数量、选客户加成档（A/B/C）、勾印刷/烫金次数。
  4. 加入报价单：点「＋ 加入报价单」，可以加多个产品拼成一整张单。
  5. 出单：右边就是报价单，可 打印 / 导出 PDF 或 复制文本 发给客户。

💡 配置产品时下面还有「兼容参考」——列出能替换当前件的兼容件（✅同套系 / 🟡同口径），点一下就切换，方便比价换件。

2 规则 · 说一句话就改

加成、数量折扣、工艺费、兼容规则——每条都是一句人话，改也是说一句话。
  1. 说一句话：在最上面的框里用大白话下指令，点「解析」。例如：
B客户加成改成28%烫金费改成0.3 2万个以上打5%折CC20945和CC20793不兼容
  1. 看预览：AI 把它翻成一条改动，给你看「改前 → 改后」，比如 B档 +30% → +28%。
  2. 确认生效：点「✔ 确认生效」才真正写入；不对就取消。AI 只提议，你拍板。

下面「现有规则」把所有规则用人话列着，每条能改 / 删；「改动记录」留痕，谁改的、改了啥、什么时候，一清二楚。

3 数据 · 上传新报价单

工厂发来新的 Excel 报价单，传上去自动读进系统。
  1. 选文件：点「选择文件」，选工厂发来的 .xlsx / .xls 报价单。
  2. 解析预览：点「解析预览」，系统读出来给你看一眼 —— 解析到多少条、哪些是新套系/新编码。
  3. 确认导入：没问题就点「✔ 确认导入」。
  4. 立刻能用：导入后，回「报价员」页刷新，新套系就能选来报价了。

🎬 花 1 分钟试一下

按这个顺序点一遍，就能看到它最厉害的地方 —— 改一条规则，报价立刻跟着变：

  1. 「规则」页 → 打「B客户加成改成28%」→ 解析 → 确认生效。
  2. 切「报价员」→ 随便配一个产品 → 客户档选 B → 看价格，已经按 28% 在算了。
  3. 回「规则」页底部「改动记录」→ 刚才那次改动有留痕。
不用登录、不用安装 —— 打开网址即用，电脑 / 手机浏览器都行。 底层是把 李记 100 份报价 Excel 整理成的一个数据库（3600+ 条价目、130+ 套系）；配套的「数据库总览」页里能看到里面有什么。 AI 副驾把人话翻成规则改动，始终要人确认才生效并留痕 —— 又灵活又安全。
